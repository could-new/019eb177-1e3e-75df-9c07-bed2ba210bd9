# स्थानीय तहमा लैंगिक समानता, अपाङ्गता र सामाजिक समावेशीकरण (GEDSI) परीक्षण

यो एप [CouldAI](https://could.ai) द्वारा सिर्जना गरिएको एक प्रस्तुतीकरण (PPT) भ्यूअर हो। यसले नेपाल सरकारको दृष्टिकोण अनुसार स्थानीय तहमा GEDSI परीक्षण किन गरिन्छ भन्ने बारे जानकारी नेपाली भाषामा प्रदान गर्दछ।

## मुख्य सुविधाहरू

- स्लाइड-आधारित प्रस्तुतीकरण इन्टरफेस
- स्थानीय तहमा GEDSI को महत्त्व बारे विस्तृत जानकारी
- अघिल्लो र अर्को स्लाइडमा जान सजिलो नेभिगेसन
- प्रतिक्रियात्मक डिजाइन (मोबाइल, ट्याब्लेट र डेस्कटपमा उपयुक्त)

## प्रयोग गरिएको प्रविधि

- Flutter
- Dart
- Material Design

## प्रोजेक्ट संरचना

```text
lib/
  main.dart      मुख्य एप फाइल र स्लाइड नेभिगेसन लजिक
pubspec.yaml     प्याकेज मेटाडेटा
```

## कसरी सुरु गर्ने

Flutter इन्स्टल गर्नुहोस्, र डिपेन्डेन्सीहरू तान्नुहोस्:

```bash
flutter pub get
```

एप चलाउनुहोस्:

```bash
flutter run
```
